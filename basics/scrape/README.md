# Scrape

This folder contains a general web-scraping practice script.

## Scripts

| Script | Purpose |
| --- | --- |
| `scrape_titles_subtitles.py` | Scrapes page titles and heading text from one or more webpages, saves JSON, and converts the result to CSV. |

## Usage

```bash
python3 basics/scrape/scrape_titles_subtitles.py https://automatetheboringstuff.com/2e/chapter11/
python3 basics/scrape/scrape_titles_subtitles.py https://example.com --output-json titles.json --output-csv titles.csv
```
